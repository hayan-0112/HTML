/**/ 
			CSS: cascading style sheet 은 HTML 문서에 색, 모양, 출력 위치 등 외관을 꾸미는 기능의 스타일 시트 언어
			CSSS로 작성된 코드를 스타일 시트라고 함.

			CSS를 이용해서 색상/배경/텍스트/폰트/박스모델/리스트 등을 수정 가능

			#CSS스타일 시트의 구성
			CSS스타일 시트 코드는 선택자{속성명:속성값;} 형태로 구성
			CSS스타일 시트 내에서 주석 처리는 /* ~~~ */ 사용
			CSS스타일 시트 내 셀렉터/프로퍼티/값은 대소문자 구분 X
			CSS작성 시 여러 속성 적용하려면 ; 으로 속성 쌍 구분

			#CSS스타일 시트를 작성하는 위치
			1. head 태그 내부에 style 태그를 작성하여 그 안에 CSS코드 작성
			2. style속성에 직접 css코드 작성
			3. 스타일 시트를 외부 파일 .css 로 작성하여 연결

			#셀렉터와 태그
			셀렉터는 CSS스타일 시트의 이름이나 규칙
			스타일을 지정하려는 대상을 지목하는 방법

			#style태그의 특징
			style태그는 여러 세트 생성 가능
			style태그에 작성된 스타일 시트는 웹 페이지 전체에 적용됨

			#CSS3 규칙
			CSS3 스타일 시트에는 규칙이 있음
			1. CSS3 스타일은 부모 태그로부터 상속 된다.
			자신을 둘러싸는 태그를 부모태그 혹은 부모요소라고 함
			부모요소에 적용 된 스타일 시트는 자식에게도 적용 됨

			2. 스타일 합치기와 오버라이딩
			브라우저는 각 태그에 대한 디폴트 스타일 시트를 가지고 있음
			개발자가 지정한 스타일이 없으면 디폴트가 적용 된다.
			따라서, 특정 태그에는 아래 4가지 상황에 따라 스타일이 적용
			1) 브라우저의 디폴트 스타일
			2) .css외부 스타일 시트 파일
			3) style태그 내에 작성된 스타일 시트 코드
			4) 태그에 직접 작성된 스타일 시트 코드
			*우선순위 1->2->3->4 : 브라우저 시트가 가장 우선순위 약함

			오버라이딩은 덮어쓰기로 동일한 우선순위에 있는 CSS시트간
			나중에 적용된 스타일이 최종 적용됨

			#id와 class 속성의 공통, 차이
			공통점: 특정 태그(요소)에 식별자를 주는 방식
			차이점: id는 문서에서 특정 하나의 요소에만 유일하게 부여해야 함
				class명은 여러 요소에 중복으로 적용 가능

			#셀렉터(선택자)의 종류
			1. 태그명 셀렉터 :태그명으로 셀렉터 지정
			2. id셀렉터 : #아이디로 셀렉터 지정
			3. class셀렉터 : .클래스명으로 셀렉터 지정
			4. 조합된 셀렉터 : 2개 이상의 셀렉터를 조합하여 태그를 구체적으로 지정 가능
				4.1: 자식 셀렉터 : child selector : 기호 > 사용 ex) div > span
				4.2: 자손 셀렉터 : descendent selector : 공백을 넣는다 ex) div span
			5. 전체셀렉터 : * 모든 요소 선택
			6. 속성셀렉터 : input 태그에 지정된 type 속성에 따라 구분하는 셀렉터
			7. 가상셀렉터 : pseudo selector 가상셀렉터는 특정 상황의 요소를 선택하는 방법으로 => 요소:상황 형태로 작성
				ex) h3:first-letter => h3의 첫 글자만 지정
				ex) li:hover => li 에 마우스를 올린 상황
				가상 셀렉터의 종류
				#hover : 마우스가 올라갈 때 적용
				#active : 마우스로 누르고 있는 상황
				#focus : 폼 요소가 마우스로 클릭해서 활성화 될 때
				#link : 방문하지 않은 링크 상황
				#visited : 방문한 링크에 적용 - visited 속성은 색상 관련 프로퍼티만 적용 가능
				#first-letter : 해당 요소의 첫 글자 (블록요소에만 적용)
				#first-line : 첫 라인에 적용 (블록요소에만 적용)
				#nth-child(even) : 짝수 번째 모든 자식 태그 지목
				#nth-child(1) : 첫 번째 자식 태그 지목
				#nth-child(2) : 두 번째 자식 태그 지목
