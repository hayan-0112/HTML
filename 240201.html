#박스의 속성 설정 display : block / inline / inline-block

			#박스의 배치 설정 position

			개발자가 작성한 순서대로 위에서 아래로 나타나는 기본 배치를
			normal flow 라고 함. 이런 기본 흐름을 무시하고 position 속성을
			이용해서 원하는 위치에 박스를 배치할 수 있다.

			position 속성에 지정 가능한 값은 아래와 같음.
			position:static
			position:relative
			position:absolute
			position:fixed

			position 속성 외에 float 속성을 통한 배치도 가능
			float:left
			float:right

			position 속성을 사용할 때, HTML 태그의 위치는 다음 프로퍼티로
			조절한다.
			top / bottom / left / right

			#static: 정적 배치 (디폴트)
			정적배치는 웹 페이지가 작성된 태그 순서대로 html태그의 출력 위치를
			정하는 방식으로, 브라우저의 디폴트 배치 방식이다.

			따라서 static의 경우에는 left / top / bottom / right / left 가 적용되지 않는다.

			#relative : 상대 배치(원래 위치를 기준으로)
			normal flow에 따라 HTML 태그가 출력되는 위치를 기준으로 
			left / top / bottom / right 값을 주면 기준위치에서 주어진 값 만큼 이동함.

			#absolute : 절대 배치((0,0)을 기준으로)
			position absolute 절대 배치를 사용하면 HTML태그를 절대위치에 배치할 수 있다.
			left/top 등 속성으로 옮기고자 하는 대상에게 absolute를 적용하고
			좌표 기준으로 설정하고자 하는 부모 컨테이너에게도 static이 아닌 position 속성을
			지정해야 함.
			부모 태그 안에서의 상대좌표.

			#fixed : 고정 배치
			브라우저를 스크롤하거나 크기를 변경해도 HTML 태그가 뷰포트(브라우저 윈도우)의
			특정 위치에 고정하는 배치 방법.
			left/top/bottom/right 프로퍼티로 뷰포트의 특정 위치 값을 지정.

			#float 배치
			브라우저의 크기를 변경하면 텍스트나 이미지가 다음 줄로 내려가는 등 위치가 변한다.
			float 프로퍼티를 지정하면 태그를 우측 혹은 좌측에 항상 배치할 수 있음.
			float은 항상 같은 위치에 출력하려는 공지 혹은 광고 창 등에 적합.

			#z-index 3차원 축 깊이
			z index 는 HTML 태그를 z축을 따라 쌓는 순서 stacking order를
			지정하는 프로퍼티로 값이 클수록 위에 놓임.
			기본 z-index를 지정하지 않은 태그들은 z-index 0으로 간주.
			z-index는 position프로퍼티가 static이 아닐 때만 작동.

			#visibility
			visibility 속성을 이용해서 HTML 태그를 보이지 않게 처리 할 수 있음.
			visibility:hidden으로 적용하면 HTML 태그에 출력공간을 할당한 채
			보이지만 않게 처리 됨.
			값: visible or hidden

			#display:none을 이용한 숨기기
			#display:none을 적용하면 마찬가지로 브라우저에서 보이지 않게 되는데,
			visibility:hidden과 차이점은 공간 할당 여부임.
			display:none은 공간 할당이 되지 않는다.

			#overflow
			overflow는 HTML 컨텐츠가 width와 height에 주어진 사이즈를 넘어가는 경우에
			컨텐츠를 어떻게 처리할 것인지 지정하는 프로퍼티로 블록태그에만 적용됨.
			.overflow 프로퍼티가 적용되려면 width와 height가 박스에 적용되어 있어야 함.

			overflow에 적용 가능한 값:
			1. visible : (디폴트)컨텐츠가 잘리지 않고 태그 영역을 넘어서 출력
			2. hidden : 넘어가는 부분은 잘라서 보이지 않음
			3. scroll : 스크롤바를 항상 부착하여 출력
			4. auto : 컨텐츠가 박스 높이보다 넘치면 스크롤바 생성

			#리스트 관련 프로퍼티
			리스트는 데이터를 나열하는 목적의 태그로 목차나 메뉴 네비게이션 바를 만들거나
			링크 모음 등 여러 용도로 활용 됨.
			list-style-type : 아이템 마커 타입 지정
			list-style-image : 아이템 마커 이미지 지정
			list-style-position : 아이템 마커 출력 위치 지정
			list-style : 위 3개 프로퍼티 한번에 지정 가능한 단축 프로퍼티명

			#마커위치 list-style-position
			inside | outside 값 지정
			outside가 디폴트 

			#마커 스타일 list-style-type
			square : 사각
			circle : 원
			upper-roman : | || |||
			lower-alpha : a b c
			none : 없음
			decimal : 1 2 3

			#list-style 단축 프로퍼티
			list-style : list-style-type, list-style-position, list-style-image 형태로 여러 값을 단축 프로퍼티로 한번에 지정
