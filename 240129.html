			#하이퍼링크 a
			웹 사이트는 여러 개의 HTML 페이지를 가지고 페이지들이 상호 연결하는 방식으로
			전체를 구성한다. HTML 페이지에서 다른 HTML 페이지로 연결하는 고리를 하이퍼링크
			라고 함. 하이퍼링크는 텍스트나 이미지에 다른 웹 주소를 달아서 만든다.
			하이퍼링크는 같은 사이트에 다른 HTML 페이지 뿐 아니라 다른 사이트에 있는 웹 페이지도
			연결 가능하다. 하이퍼링크를 따라서 다른 웹 페이지로 이동하는 것을 네비게이션이라 부름
			혹은 링크라고 줄여서 표현.

			#하이퍼링크 a 사용방법
			1. a 태그는 여는 태그와 닫는 태그 모두 작성
			2. a 태그는 하이퍼링크 목적의 태그로 이동하려는 페이지의 주소를 필수 작성
			3. href속성에 URL 입력
			1111			
			1번구역
			2번구역
			3번구역
			id를 통해 이동하는 앵커 기능
			href에 #id값으로 이동 위치 지정 가능

			#인라인 프레임 iframe
			iframe태그를 사용해서 현재 HTML 페이지 내 내장 윈도우를 만들고 다른 HTML페이지를 출력할 수 있다.
			iframe태그를 사용해서 만들어지는 윈도우를 인라인 프레임 inline-frame이라고 함.
			

			iframe 태그의 속성
			1. src : 출력하려는 url 주소
			2. name : 프레임 윈도우의 이름 속성: name 속성은 다른 웹 페이지에서 target 속성의 대상으로 지정 가능
			3. width
			4. height
			
			#iframe으로 출력이 불가능한 URL
			-네이버 외 대부분의 포탈 사이트는 iframe으로 연결할 수 없도록 방지 처리 해놓음
			-Clickjacking 웹 침입 방지 설정이 되어 있음

			#iframe 사용시 주의
			1. iframe은 body 태그 내에서만 사용
			2. src 속성은 필수 입력
			3. 인라인 프레임 안에 또 다른 인라인 프레임을 중첩 가능
			
			#브라우저 윈도우와 인라인 프레임 윈도우의 계층 관계
			브라우저 윈도우와 그 안에 생성된 인라인 프레임 윈도우는 부모-자식 관계의 계층 관계 형성
			인라인 프레임이 또 다른 인라인 프레임을 포함하면 이것도 부모-자식 관계 형성
			1. parent 윈도우: 부모 윈도우 속성
			2. child 윈도우 : 자식 윈도우 속성
			3. top 윈도우 : 최상위 윈도우 의미 속성

			윈도우가 여러 중첩 계층 형성하고 있을 때, top, parent, child 속성을 통해 특정 윈도우 지정 가능
