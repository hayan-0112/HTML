#JS의 특징
            1. 자바스크립트는 소스코드 형태로 HTML페이지에 내장됨
            2. 자바스크립트 소스코드는 컴파일 과정 없이 브라우저 내부 자바스크립트 처리기(인터프리터)에 의해 바로 실행
            3. 자바스크립트는 C언어 구조를 차용하고 단순화 해놓음

            #웹페이지에서 자바스크립트의 역할
            1. 동적인 구현을 자바스크립트로 한다
            2. HTML 태그로 존재하는 요소들을 전부 객체화할 수 있다.
            3. 사용자의 입력, 계산
            4. HTML 구조의 변경, 제어
            5. 브라우저 제어
            6. 웹 서버와 통신
            7. 웹 애플리케이션 개발
                HTML5는 캔버스, 로컬 및 세션, 위치 정보 등 자바스크립트 언어로 활용 할 수 있는 많은 API를 제공

            #자바스크립트 코드의 위치
            1. HTML 태그의 이벤트 리스너 속성에 작성
            2. script 태그 내에 작성
            3. 외부 JS 파일에 작성
