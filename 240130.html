검색 버튼을 누르면 브라우저는 form태그의 action에 지정 된 주소를 참고해서, 해당 서버에 접속 후 search.naver 응용 프로그램의 실행을 요구 사용자가 입력 창 input에 텍스트 입력 후 검색 누르면, 브라우저는 웹 서버 응용 프로그램에 보낼 폼 데이터를 만든다. query=검색어 여기서 query는 입력창의 이름인 name속성에 지정된 값. 폼 데이터는 name=value 형태로 구성된다. form태그의 method 속성이 get으로 설정되어 있고 get은 폼데이터를 action의 url에 덧붙인다.

input type 속성의 값 종류 1. text: 텍스트 입력 창 2. password: 비밀번호 3. button: 일반 버튼 4. submit: 웹 서버로 폼 데이터 전송하는 버튼 5. reset: 입력된 폼 데이터 초기화 버튼 6. image: 이미지 버튼 7. checkbox: 체크박스 8. radio: 라디오 버튼 9. month / week / date / time / datetime-local : 년 월 일 시간 등 시간 입력 창 10. number / range: 스핀 버튼과 슬라이드바로 숫자 입력 창 11. email / url / tel / search: 이메일, 주소, 전화번호, 검색키워드 등 형식 검사 기능 창 12. file: 로컬 컴퓨터 파일 선택 창 #input태그가 아닌 태그로 작성하는 입력 창 종류 textarea : 여러줄 텍스트 입력창 cols, rows: 텍스트 입력창 크기 중 가로 세로 문자 수 wrap: 자동 줄바꿈 처리 2.datalist
